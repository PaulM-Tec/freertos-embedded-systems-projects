## Overview
This repository contains a collection of embedded systems projects developed using **FreeRTOS and C** as part of an Honours-level Embedded Systems Software Development module.

The projects demonstrate real-time system design, concurrent task execution, synchronization mechanisms, and secure system architecture using the **WIN32 FreeRTOS simulator in Visual Studio**.

---

## Project Structure

### Traffic Light Control System
Location:

A real-time traffic management system designed for a busy intersection.

**Features:**
- Control of North–South and East–West traffic lights
- Pedestrian signal synchronization
- Emergency vehicle detection and preemption
- Real-time console simulation

**Key Concepts:**
- FreeRTOS task scheduling
- Event-driven system behaviour
- Preemptive multitasking

---

### Weather Station System
Location:

A multitasking embedded system for monitoring environmental conditions.

**Features:**
- Temperature monitoring every 5 seconds
- Humidity monitoring every 7 seconds
- Data logging every 10 seconds
- Real-time display every 2 seconds

**Key Concepts:**
- Concurrent task execution
- Mutex synchronization
- Shared resource protection
- Real-time data logging
